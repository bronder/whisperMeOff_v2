# Release Notes

## v1.0.0 (Current)

### Features
- Global hotkey recording (`Ctrl+Shift+R`)
- Real-time VU meter visualization
- Waveform visualizer
- System tray integration
- GPU-accelerated transcription (Vulkan/AMD)
- Custom model file selection
- Multi-language support

### UI Improvements
- Compact main window design
- Settings window without menu bar
- Dark theme throughout

### Known Issues
- None reported

---

## Installation

1. Download the latest release
2. Run `whisperMeOff.exe`
3. Configure your Whisper model path in Settings

### Prerequisites
- FFmpeg must be in your PATH for audio conversion
- Whisper model file (.bin) required for transcription
